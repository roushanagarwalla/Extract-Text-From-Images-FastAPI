# Extract-Text-From-Images-FastAPI

An API build using FastAPI for extracting text from images.

**Languages/Frameworks Used:** Python, FastAPI, Pytesseract

**Other tools used:** Docker, Heroku(For Hosting)

The live version of the API can be accessed by https://text-images.herokuapp.com/

To See the **Docs** of the API generated by Swagger UI check the url https://text-images.herokuapp.com/docs
